### 1. Prerequisites

Ensure you have the following installed on your system:

- **Node.js** (Recommended Version: LTS)
- **Git**

### 2. Local Setup

**a. Clone the repository:**

## `git clone [https://github.com/KHOA0302/smarthome_be.git](https://github.com/KHOA0302/smarthome_be.git)`

## `cd smarthome_be`

**b. Install dependencies:**

All required libraries are installed using the following command:

## `npm install`

## Running the Application

You can use the predefined scripts in package.json to run the project.

### 1. Development Mode (Recommended)

This command starts both the Server and the Worker concurrently, which is ideal for local development.

## `npm run dev:all`

### 2. Production Mode

Run the Server process in a standard production environment:

## `npm start`
